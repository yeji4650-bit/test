로또 번호 추첨기 사용 방법

  1. 추첨하기 버튼을 클릭합니다.
  2. 1~45 사이의 번호 중 6개의 당첨 번호와 1개의 보너스 번호가 자동으로 생성됩니다.
  3. 5게임의 번호가 동시에 생성되므로 원하는 번호 조합을 선택하세요.
  4. 생성된 번호를 로또 용지에 직접 마킹하거나 기억해 두세요.

로또(Lotto) 란?

로또 6/45는 1부터 45까지의 숫자 중 6개를 선택하는 대한민국 복권입니다. 매주 토요일 저녁 추첨이 진행되며, 1등 당첨 번호 6개를 모두 맞혀야 1등에 당첨됩니다. 보너스 번호는 2등 당첨 여부를 결정하는 데 사용됩니다.

번호 색상 안내

1–10 11–20 21–30 31–40 41–45
